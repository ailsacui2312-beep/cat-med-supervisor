## 指挥官派发（来自 dev-playbook 总指挥 session）

> 以下任务由"指挥官"根据技术扫描和跨项目分析自动写入，启动时请评估并处理。

### 2026-03-25: 评估升级 Expo SDK 55
- **来源**: 2026 Q1 tech-scout 发现
- **内容**: Expo SDK 55 冷启动 +43%，OTA 体积缩小 75%。CatMed 目前是 SDK 54
- **行动**: 家庭共享做完后评估升级。重点关注长辈模式下的启动速度提升
- **优先级**: 低 — 家庭共享优先

### 2026-03-25: Lottie 替代视频背景
- **来源**: tech-opportunities.md 已有记录
- **内容**: 猫咪视频背景有颜色适配问题，Lottie 矢量动画可解决（体积小、透明背景、帧率流畅）
- **行动**: 找 1-2 个猫咪主题 Lottie 动画测试替换效果
- **优先级**: 中

### 2026-04-01: ⚠️ Axios NPM 供应链污染安全检查
- **来源**: tech-scout 2026-04-01，HN 1889分重大安全事件
- **内容**: axios 在 npm 上被投放恶意版本，携带 RAT（远程访问木马）
- **行动**: (1) 检查 package.json 中 axios 版本号；(2) 运行 `npm audit`；(3) 与 RLS 安全审查合并做，上线前必须完成
- **优先级**: 高 — 立即检查

### 2026-03-31: 家庭共享 RLS 上线前安全审查
- **来源**: tech-scout 2026-03-31，Lovable 安全研究报告
- **内容**: 研究发现 10.3% 的 AI 生成 Supabase 应用有关键 RLS 配置漏洞。CatMed 家庭共享的 RLS 是 Claude 写的（递归 bug 已修复），但不应假设 AI 生成的 RLS 一定安全
- **行动**: 家庭共享功能上线前，专门做一次 RLS 安全测试：（1）用普通用户账号尝试越权访问其他家庭的猫咪数据，（2）确认 RLS 政策覆盖了所有 CRUD 操作
- **优先级**: 高 — 家庭共享前端做完后必须执行，不能跳过
